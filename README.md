## R 程式設計班課程

### 簡介

這是[郭耀仁](https://www.facebook.com/yaojen.kuo.1)在[台灣大學資訊工程學系資訊系統訓練班](https://www.csie.ntu.edu.tw/train/)所開設的 R 程式設計班課程網頁，課程時數 30 小時。

### 評分標準

- 45% 出席率
- 25% 期中作業
- 30% 期末作業
